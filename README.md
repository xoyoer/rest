# 休息

**帮你关心的人按时休息。**

一个 macOS 菜单栏应用。当你关心的人连续工作太久，它会温柔但坚定地让 TA 停下来。

---

## 为什么做这个

有些人工作起来会忘记一切。连续三四个小时不喝水、不站起来、不看窗外。他们不是不知道该休息，只是太专注了。

这个应用不是给工作狂自己用的 —— 是给**关心工作狂的人**用的。

你的伴侣、父母、朋友，发现了这个应用，安装到 TA 的电脑上，设置一个只有自己知道的密码。从此，TA 想退出程序？要密码。想跳过休息？得按住 5 秒。想用 Cmd+Q 偷偷关掉？拦截了。

不是控制，是守护。

---

## 功能

**核心机制**
- 连续工作到设定时长后，全屏进入休息
- 休息屏幕覆盖所有窗口，显示名言和呼吸动画
- 屏幕渐暗进入，像走进一个安静的房间
- 无键鼠活动 3 分钟自动暂停计时

**守护锁**
- 由关心你的人设置密码
- 退出程序需要密码
- Cmd+Q 被拦截
- 菜单栏隐藏退出按钮
- 首次启动引导家人设置

**跳过保护**
- 每天最多跳过 1 次
- 需要长按 5 秒才能跳过（不是点一下）
- 长按时显示反思文字
- 跳过按钮默认隐藏，鼠标移动才出现

**日常使用**
- 菜单栏叶子图标，安静运行
- 疲劳度百分比、工作时长、休息倒计时
- 今日工作时间线
- 历史数据图表
- 开机自动启动（不可关闭）

---

## 休息屏幕

进入休息时，屏幕缓慢变暗，像走进一个安静的房间。

- 当前时间（锁屏风格）
- 随机中英文名言
- 呼吸节奏动画（遮罩层缓慢明暗变化）
- 极简倒计时
- 多屏幕支持

## 安装

### 下载

从 [Releases](../../releases) 页面下载最新的 DMG 文件。

### 首次打开

由于没有 Apple 开发者签名，首次打开需要：

1. 打开 DMG，将「休息」拖入 Applications
2. 在 Applications 中找到「休息」
3. **右键点击** → 选择「打开」
4. 弹出提示后点「打开」

只需要第一次这样操作。

### 系统要求

- macOS 14.0 (Sonoma) 或更高版本
- Apple Silicon (M1/M2/M3/M4)

## 技术

- Swift + SwiftUI + AppKit
- 无 Xcode 依赖，纯 `swiftc` 编译
- 数据存储在本机，不联网
- 开源，MIT 协议

### 自行编译

```bash
git clone https://github.com/xoyoer/rest.git
cd rest
bash build.sh
open build/休息.app
```

## 设计理念

> 这个产品最重要的时刻不是设置，不是菜单栏，不是跳过流程 —— 是那个屏幕变暗、世界安静下来的瞬间。

如果那个瞬间做对了，让工作狂在前 3 秒就感到"也许停下来也不错"，后面所有的防线都不重要了。
