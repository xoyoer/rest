# 休息

**密码不在 TA 手里。**

市面上的休息提醒都有同一个问题：关掉就没了。弹窗可以点掉，倒计时可以跳过，程序可以退出。靠自觉管用的话，就不需要提醒了。

这个 app 换了一个思路 — 密码在关心 TA 的人手里。TA 退不掉，关不了，跳不过。开机自动启动，到点全屏休息。唯一能解锁的人是你。

一个 macOS 菜单栏应用。给关心工作狂的人用，也给知道自己该休息但停不下来的人用。

---

## 和其他休息提醒的区别

其他 app：提醒你 → 你点掉 → 继续工作 → 提醒失败

这个 app：到时间 → 全屏锁定 → 没有跳过按钮 → 想退出要密码 → 密码在别人手里

核心差异就一个：**控制权不在使用者手上。**

不设置守护锁也能用 — 到点照样全屏休息，只是退出程序不需要密码（有 30 秒冷却）。适合自律够用、只需要一个强制全屏的人。

---

## 功能

- 连续工作到设定时长，全屏进入休息。不可跳过，不可关闭
- 守护锁：密码由家人设置，退出程序需要密码，Cmd+Q 被拦截
- 开机自动启动，关不掉
- 休息屏幕：桌面壁纸渐暗 + 呼吸节奏 + 随机名言
- 无键鼠活动自动暂停计时，不浪费你离开的时间
- 工作时间线：每段工作和休息都清晰记录，像一份今天做了什么的清单
- 历史图表：按天查看工作时长和休息次数
- 所有数据仅存本机

## 安装

### 下载

从 [Releases](../../releases) 页面下载最新的 DMG 文件。

### 使用方式

1. 打开 DMG，将「休息」拖入 Applications
2. 双击打开，系统会提示无法验证 — 点「完成」
3. 打开 **系统设置 → 隐私与安全性**，往下滚找到「已阻止使用"休息.app"」，点 **仍要打开**
4. 首次启动会自动打开设置，在设置中启用守护锁
5. 完成。TA 不会注意到它，直到该休息的时候

### 系统要求

- macOS 14.0 (Sonoma) 或更高版本
- Apple Silicon (M1/M2/M3/M4)

## 编译

```bash
git clone https://github.com/xoyoer/rest.git
cd rest
bash build.sh
open build/休息.app
```

无 Xcode 依赖，纯 `swiftc` 编译。
