# 休息

**帮你关心的人按时休息。**

一个 macOS 菜单栏应用。给关心工作狂的人用，也给知道自己该休息但停不下来的人用。

---

## 故事

有些人工作起来会忘记一切。连续三四个小时不喝水、不站起来、不看窗外。

你提醒 TA？TA 说"马上"。你再提醒？TA 说"别烦我"。你不提醒？你担心。

**口头提醒 = 唠叨 → 关系紧张。** 越关心，冲突越多。

这个应用把"唠叨"变成一个安静的程序，把"冲突"变成一次安装。

你装到 TA 的电脑上，设一个只有你知道的密码。从此 TA 每工作一段时间，屏幕就会变暗，温柔但坚定地让 TA 停下来。TA 想退出？要你的密码。想关掉？Cmd+Q 拦截了。想重启不开？开机自动启动。

**不是控制，是守护。**

---

## 功能

**休息屏幕**
- 连续工作到设定时长后，全屏进入休息
- 屏幕渐暗，像走进一个安静的房间
- 呼吸节奏动画，引导放松
- 锁屏风格的时间显示
- 随机中英文名言
- 不可跳过，不可关闭

**守护锁**
- 在设置中启用，退出程序需要密码
- Cmd+Q 被拦截
- 菜单栏隐藏退出按钮
- 密码在家人手里

**系统**
- 开机自动启动（不可关闭）
- 无键鼠活动自动暂停计时
- 工作时间线 + 历史图表
- 所有数据仅存本机

## 安装

### 下载

从 [Releases](../../releases) 页面下载最新的 DMG 文件。

### 使用方式

1. 打开 DMG，将「休息」拖入 Applications
2. 首次打开：**右键 → 打开**（自签名需手动允许一次）
3. 在设置中启用守护锁，设置密码
4. 完成。TA 不会注意到它，直到该休息的时候

### 系统要求

- macOS 14.0 (Sonoma) 或更高版本
- Apple Silicon (M1/M2/M3/M4)

## 编译
